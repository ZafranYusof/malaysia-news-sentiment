============================================================
INFINITE LOOP PREVENTION
============================================================

Key Fix (Line 309-314):
Before: useEffect(..., [articles, ...])
After: useEffect(..., [articles.length, ...])

Reason: Articles array reference changes on every data fetch,
causing the effect to re-trigger indefinitely. Using length
prevents this while still detecting new data.
